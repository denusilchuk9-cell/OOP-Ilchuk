# Рефакторинг коду курсового проекту "Електронний журнал"

## Мета роботи
Виявлення та усунення code smells у коді курсового проекту.

---

## Знайдені Code Smells

### 1. Long Method (Занадто довгий метод)
- **Місце:** GradeForm.cs, метод LoadData()
- **Проблема:** метод виконував 5 різних дій одночасно
- **Рішення:** розбито на 5 окремих методів

---

### 2. Duplicate Code (Дублювання коду)
- **Місце:** AttendanceForm.cs та GradeForm.cs
- **Проблема:** однаковий код завантаження студентів та предметів у двох формах
- **Рішення:** створено спільний клас FormHelper.cs

---

### 3. Magic Numbers (Магічні числа)
- **Місце:** Grade.cs, метод IsPassing()
- **Проблема:** число 60 без пояснення
- **Рішення:** замінено на константу PassingThreshold

---

## Результат
- Всі 34 тести пройшли успішно
- Код став більш читабельним та підтримуваним

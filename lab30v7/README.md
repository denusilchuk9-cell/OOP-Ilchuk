# Лабораторна робота №30
## Тема: Написання юніт-тестів з xUnit

### Мета роботи
Навчитися писати юніт-тести для власного коду за допомогою xUnit, використовуючи різні типи assertion та параметризовані тести.

### Варіант завдання
Варіант 7 – ShoppingCart (кошик для покупок)

### Хід роботи

#### 1. Створення проєктів
Створено основний проєкт `lab30v7` та тестовий проєкт `lab30v7.Tests` з використанням xUnit Framework.

#### 2. Реалізація класів
Розроблено два класи:
- **CartItem** – представляє товар у кошику (назва, ціна, кількість)
- **ShoppingCart** – реалізує бізнес-логіку кошика з методами:
  - `AddItem()` – додавання товару
  - `RemoveItem()` – видалення товару
  - `CalculateTotal()` – розрахунок загальної вартості
  - `Clear()` – очищення кошика

#### 3. Написання юніт-тестів
Створено 16 тестів, серед яких:
- **8 тестів з атрибутом [Fact]** – перевіряють окремі сценарії
- **3 тести з атрибутом [Theory]** – параметризовані тести з різними вхідними даними

Тести покривають:
- Успішні сценарії (додавання, видалення, розрахунок суми)
- Помилкові сценарії (некоректна кількість товару)
- Крайні випадки (порожній кошик, видалення відсутнього товару)
